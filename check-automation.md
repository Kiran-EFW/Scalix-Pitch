2. **Check Workflow Permissions:**
   - Go to: Settings ‚Üí Actions ‚Üí General
   - Ensure "Workflow permissions" allows read/write

3. **Check Recent Runs:**
   - Go to: Actions tab
   - Click on failed runs to see error messages

4. **Verify Firebase Project:**
   ```bash
   firebase projects:list
   ```
